The Requirements

Each templating engine should be able to:

  • Render the complete file
  • Include sub templates
  • Escape and not evaluate an injected JavaScript string

The Goal

Assessment will be based on the following:

  • Performance, performance, performance...
  • Ease of implementation & aesthetics of syntax
  • Flexibility
  • File size
